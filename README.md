# Game UI Component Library

A responsive Game UI Component Library built using HTML, CSS, and JavaScript.

## Features

- Reusable Buttons
- Cards
- Progress Bars
- Input Fields
- Toggle Switches
- Modal Popup
- Responsive Layout

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Project Structure

```
game-ui-library/
│── index.html
│── style.css
│── script.js
└── README.md
```

## Purpose

This project demonstrates reusable game UI components commonly used in modern game interfaces.
